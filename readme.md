# 🚇 TransitRadar Berlin

TransitRadar Berlin is a web application that visualizes Berlin public transport stations on an interactive map using Leaflet.js and live BVG transport data.

Users can search for stations, view nearby stops, and see upcoming departures directly on the map.

## Features

* Interactive map powered by Leaflet.js
* Dark theme using Carto Dark Matter
* Station search with live suggestions
* Automatic loading of visible stations
* Grouped stations with the same name
* Live departure information in station popups
* Modular JavaScript structure
* Responsive map navigation and zooming

## Technologies

* HTML5
* CSS3
* JavaScript (ES Modules)
* Leaflet.js
* BVG Transport REST API
